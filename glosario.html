HTML5

es la última versión de HTML. El término representa dos conceptos diferentes: Se trata de una nueva versión de HTML, con nuevos elementos, atributos y comportamientos. Contiene un conjunto más amplio de tecnologías que permite a los sitios Web y a las aplicaciones ser más diversas y de gran alcance. A este conjunto se le llama HTML5 y amigos, a menudo reducido a HTML5 .

Addres

Muestra la información de contacto de una persona, como por ejemplo, nombre y dirección de correo electrónico, es decir, con este tag, podemos especificar información de contacto relevante sobre la persona, empresa que realiza una web, pero su utilización no está pensada para una dirección física de un lugar, sino más bien para la datos de contacto online.

Article

Este elemento es uno de los nuevos tags de HTML 5 que nos otorga la posibilidad de dar un significado a ciertos contenidos o grupos de contenidos en la web, como por ejemplo, noticias o comentarios.

Aside

Nos permite establecer una relación entre contenidos similares. Cuando vemos un tag de este tipo, sabemos que el contenido que se encuentra en su interior, se relaciona con otro contenido de la misma página.

Body

Este elemento define el cuerpo principal del documento HTML que estamos editando. En HTML 4 este tag tenía la posibilidad de especificar atributos, pero estaban en desuso, mientras que en HTML 5 han sido definitivamente eliminados, por lo que el tag ya no tiene ningún atributo posible.

Head

Este elemento, define la cabecera de nuestro documento HTML, donde podemos hacer referencias a todos los scripts, hojas de estilos, titulo del documento, datos para buscadores, etc. En HTML 4 nos permitía añadir un atributo llamado profile, que nos daba la posibilidad de introducir una especificación concreta sobre la información que se indica a través de los elementos meta para los buscadores de una manera más precisa, con una url más específica que haga referencia por ejemplo a un pais. En HTML 5 el atributo profile esta en desuso Un ejemplo de esto sería el siguiente.

Header

El tag header nos permite definir contenidos como preámbulo, que es muy útil en casos como una noticia, un documento de texto o un artítulo periodístico, así como una entrada de blog. Nos permite insertar texto que describa brevemente el contenido sobre el que va a tratar dicho documento, noticia o entrada. En definitiva, nos permite que un contenido determinado se pueda diferenciar entre la cabecera, que se definiría con este tag y el cuerpo, que por ejemplo se podría definir con un tag article.

iframe

Este tag es un contenedor que nos permite cargar otra página web en una parte de la nuestra. Esta etiqueta conlleva ciertos riesgos de seguridad, ya que estamos cargando contenido ajeno al nuestro que no podemos controlar.

object

Es el tag que nos permite insertar código de otros medios externos como pueden ser flash, audio, video, aplicaciones Java, etc. Tambien se puede utilizar este tag en lugar de iframe para insertar otra web como parte de la que se está ejecutando.
